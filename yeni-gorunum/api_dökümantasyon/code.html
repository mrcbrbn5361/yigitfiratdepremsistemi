Live Earthquake Feed

Get the most recent seismic activity data.

GET /api/v1/earthquakes/live

Query Parameters

Parameter Type Description
min_magnitude float Minimum magnitude filter (e.g. 3.5).
limit integer Number of results to return. Default 20.
city string Filter by specific city name.
info

Note: Data is updated every 60 seconds from major seismic observatories including KOERI and AFAD.

Historical Data

Access earthquake archives filtered by date range.

GET /api/v1/earthquakes/history

Request

start_date ISO 8601 (YYYY-MM-DD)
end_date ISO 8601 (YYYY-MM-DD)

Sample cURL

curl -X GET "https://api.miracbirben.com/v1/earthquakes/history?start_date=2023-01-01&end_date=2023-02-01" \ -H "Authorization: Bearer YOUR_API_KEY"
