# Product Overview

This workspace is configured for AI-assisted development with Kiro. The project follows intentional minimalism principles and full-spectrum engineering practices.

## Core Principles

- **Intentional Minimalism**: Every line of code must have a purpose
- **Production-First**: All code should be production-ready from the start
- **Zero Fluff**: Direct, focused solutions without unnecessary explanations
- **Link Intelligence**: External APIs and services are treated as first-class integration candidates

## Development Philosophy

- Follow established ecosystem standards
- Prioritize readability and maintainability
- Implement proper error handling and edge cases
- Consider accessibility (WCAG AAA when applicable)
- Plan for scalability from day one
