# Technology Stack

## Development Environment

- **Platform**: Windows (win32)
- **Shell**: cmd/PowerShell
- **AI Assistant**: Kiro IDE integration

## Language Support

This workspace supports full-spectrum development across:

- **Frontend**: HTML, CSS, JavaScript/TypeScript, React, Vue, Svelte
- **Backend**: Node.js, Python, Go, Java, Rust, PHP
- **Mobile**: Swift, Kotlin, Flutter, React Native
- **Systems**: C, C++, Zig, Assembly, WebAssembly
- **Functional**: Haskell, Elixir, Erlang, OCaml
- **Data/AI**: Python, R, SQL
- **DevOps**: Docker, Kubernetes, CI/CD

## Common Commands

### Windows PowerShell
```powershell
# File operations
Get-ChildItem                    # List files
Copy-Item source dest           # Copy files
Remove-Item file.txt            # Delete files
New-Item -ItemType Directory    # Create directory

# Development
npm install; npm run build      # Node.js projects
pip install -r requirements.txt # Python projects
go mod tidy; go build          # Go projects
cargo build --release         # Rust projects
```
