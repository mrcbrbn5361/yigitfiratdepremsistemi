Yiğit Fırat Deprem Sistemi API Dokümantasyonu

Yiğit Fırat Deprem Sistemi API'sine hoş geldiniz. KOERI (Kandilli Rasathanesi) verilerini kullanan REST API'miz aracılığıyla Türkiye ve çevre bölgeler için gerçek zamanlı sismik verilere erişin. API tamamen ücretsizdir ve kimlik doğrulama gerektirmez.

bolt

Gerçek Zamanlı Veri

KOERI sensörlerinden doğrudan 5 dakikalık güncellemeler.

public

Ücretsiz Erişim

API anahtarı gerektirmez, tamamen açık erişim.

language

CORS Aktif

Tüm domainlerden erişim izni, web uygulamalarınızda kullanın.

Kandilli Deprem Verisi

KOERI Kandilli verilerini formatlanmış şekilde alın.

GET /api/earthquakes-kandilli

Sorgu Parametreleri

Parametre Tip Açıklama
limit integer Döndürülecek sonuç sayısı. Varsayılan 10, maksimum 50.
info

Veri Kaynağı: Veriler KOERI (Kandilli Rasathanesi) Orhan Aydoğdu API'si üzerinden gerçek zamanlı olarak alınır.

language

CORS Aktif: API tüm domainlerden erişime açıktır. Web uygulamalarınızda, mobil uygulamalarınızda veya herhangi bir platformda kullanabilirsiniz.

Ham Kandilli Verisi

Orhan Aydoğdu API'sinden doğrudan ham veri alın.

GET /api/kandilli-proxy

Örnek İstek

curl -X GET "https://miracbirbendepremsistemi.vercel.app/api/kandilli-proxy"

Yanıt Formatı

Orhan Aydoğdu API'sinin orijinal formatında veri döndürür. Son 50 depremi içerir.

status: true
result: [...earthquakes]

code Kullanım Örnekleri

JS JavaScript / Fetch API 

fetch('https://miracbirbendepremsistemi.vercel.app/api/earthquakes-kandilli?limit=10')
  .then(response => response.json())
  .then(data => {
    console.log('Depremler:', data.data);
  })
  .catch(error => console.error(error));

$ jQuery / AJAX 

$.ajax({
  url: 'https://miracbirbendepremsistemi.vercel.app/api/earthquakes-kandilli',
  method: 'GET',
  data: { limit: 10 },
  success: function(data) {
    console.log(data.data);
  }
});

🐍 Python / Requests 

import requests

response = requests.get(
    'https://miracbirbendepremsistemi.vercel.app/api/earthquakes-kandilli',
    params={'limit': 10}
)
data = response.json()
print(data['data'])

$ cURL / Terminal 

curl -X GET \
  "https://miracbirbendepremsistemi.vercel.app/api/earthquakes-kandilli?limit=10" \
  -H "Accept: application/json"

tips_and_updates Önemli Notlar

  • API tamamen ücretsizdir ve API anahtarı gerektirmez
  • CORS tüm domainler için aktiftir - herhangi bir web sitesinden kullanabilirsiniz
  • Rate limit: Dakikada 100, saatte 1000, günde 10.000 istek
  • Veriler 5 dakikada bir KOERI'den güncellenir
  • Tüm zaman damgaları Türkiye saati (UTC+3) ile verilir

curl -X GET "https://miracbirbendepremsistemi.vercel.app/api/test-kandilli"

Yanıt Alanları

success API durumu
status HTTP durum kodu
dataLength Veri sayısı

warning Hatalar & Limitler

Kod Açıklama Çözüm
429 Çok Fazla İstek Rate limitinizi aştınız. Dakikada 100 istek sınırı.
500 Sunucu Hatası Sunucumuzda bir sorun oluştu. Daha sonra tekrar deneyin.
503 Hizmet Kullanılamıyor Kandilli veri kaynağına erişilemiyor. Geçici bir durum.

Rate Limiting

  • • Dakikada maksimum 100 istek
  • • Saatte maksimum 1000 istek
  • • Günde maksimum 10.000 istek
  • • Aşım durumunda 429 HTTP kodu döner
